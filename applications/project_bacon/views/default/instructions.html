Welcome

Welcome to the Game of Bacon, a word guessing game. The goal is to guess a key word that makes a “path” of words from the key word to the target word by searching and linking words that have one letter difference from the previous linked word to the key word and the target word. This game is a variation of the Oracle of Bacon. This game compares the characters of two strings to link words from the key word to the target word instead of actors and movies.

How to Play

In order to play, users must log in or sign up. Once logged in, users are able to create a game by selecting a target word from the word list. To play, select a made game on the board page. While on a game page, users select and submit a key word. Upon submission the key word makes a path to the target word.
